Основи мови запитів SQL

Призначення, основні поняття та терміни мови SQL

Призначення SQL

SQL (Structured Query Language) — це мова, яка використовується для роботи з базами даних. Вона дозволяє створювати, змінювати, видаляти та отримувати дані.

Основні поняття та терміни

  • Таблиця: структура для зберігання даних у рядках та стовпцях.
  • Запит: команда для взаємодії з базою даних.
  • Стовпець: атрибут таблиці, який визначає тип даних.
  • Рядок: одиниця даних, що містить значення для кожного стовпця.

Найпростіші запити SQL

Ось приклад простого запиту для отримання даних:

SELECT * FROM Таблиця;

Запити з умовою

Для відбору даних з умовою використовується ключове слово WHERE:

SELECT * FROM Таблиця WHERE Стовпець = 'Значення';

Групування запитів

Групування даних дозволяє об'єднувати рядки на основі значень стовпців за допомогою GROUP BY:

SELECT Стовпець, COUNT(*) FROM Таблиця GROUP BY Стовпець;
